# @cool/helper

工具集军火库，集中管理工具函数

## 安装

```bash
pnpm i @cool/helper
```

## 使用

### ESM 导入

```ts
import { isNumber } from '@cool/helper'

isNumber('123')
```

### CJS 导入

```ts
const utils = require('@cool/helper')

utils.isNumber('123')
```
